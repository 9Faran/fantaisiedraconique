# Phare de Iphemedia
---
#city  
Fait partie du dossier [[Les Pays/Îles australes/Îles australes\|Îles australes]]

-------
## Description
Le Phare dédié à [[Les Dieux/Dieux des Phares/Iphemedia\|Iphemedia]] se trouve sur l'île de [[Les Pays/Îles australes/Sebaid\|Sebaid]].
## Lieux importants
Le Phare lui-même.
## Architecture
Grande construction de pierres. La tour du phare est elle-même entourée d'annexe utilisée par les prêtres.
## Routes
Une ancienne route conduit jusqu'à [[Les Pays/Îles australes/Sebaid\|Sebaid]].
## Population
Moins d'une centaine de personnes.
## Habitants
- Le Haut-Prêtre [[Les PNJs/Membre du Vieux Culte/Ipun\|Ipun]] du [[Les Guildes/Vieux Culte\|Vieux Culte]].
