# Le Pendu
---
#Items 
Fait partie du dossier [[Les Livres/Tarot/Tarot\|Tarot]]

-------
Le Pendu est le douzième arcane majeur, il représente le ressentiment, la sensation d'être piégé.
Sur cette carte, apparaît [[Cain\|Cain]] devant la potence, il tient dans sa main un miroir.
